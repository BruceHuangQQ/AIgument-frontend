# ai-debate-frontend-unihack

You can access the backend repository here: https://github.com/agamv25/AIgument-backend

Setup and run the backend server first before moving on to the next instructions

## Setup & run

1. **Install pnpm** (if you don't have it):
   ```bash
   npm install -g pnpm
   ```

2. **Install dependencies:**
   ```bash
   pnpm i
   ```

3. **Start the dev server:**
   ```bash
   pnpm dev
   ```
